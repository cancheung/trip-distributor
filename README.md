# trip-distributor

> 旅行创新供销商系统

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

```bash
### 目录结构
src
├─api                     # 接口目录
├─components              # 公共组件目录
├─filter                  # 公共过滤器目录
├─router                  # 路由目录
├─store                   # 全局数据仓库目录
├─styles                  # 样式目录
├─utils                   # 工具目录
└─views                   # 视图目录
    └─login                 # 具体页面视图
        └─components          # 视图组件
```
